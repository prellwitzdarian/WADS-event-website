# WADS-event-website

Assignment: Event Center Website; create event website. For Love and Deepspace in Seattle event group. They currently have no existing website for their event group so I created one for them.

In order to view my website, first clone my repository to your local machine. Git repository link is https://github.com/prellwitzdarian/WADS-event-website.git

Then, open up index.html in VS Code, then right click in the editor and select "open with live view".
